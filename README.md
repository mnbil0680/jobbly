## Description
Jobbly is a Single-Page Application (SPA) that helps users
search, track, and manage their job applications in one place.

## Technologies Used
- PHP
- MySQL
- AJAX (Fetch API)
- Third-Party API: TODO (e.g. Job Search API)

## File Structure
- index.php          → Main SPA page
- header.php         → Site header
- footer.php         → Site footer
- DB_Ops.php         → Database operations
- API_Ops.php        → Third-Party API connection
- API_Ops.js         → AJAX calls for API
- Upload.php         → File upload handler
- main.js            → Main JavaScript
- style.css          → All styles
- config.php         → App configuration (NOT on GitHub)
- config.example.php → Template for config.php
